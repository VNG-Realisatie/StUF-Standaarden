Er kan geen functionele bevestiging worden verstuurd Bv01

Er is een Bv03 methode opgenomen in de ZKN0310 wsdl, waarbij de request een Bv03 is en de response een Bv03, wat een onzinige combinatie lijkt te zijn. De methodenaam en Request zouden waarschijnlijk Bv01 moeten zijn, zodat er een functionele bevestiging kan worden verstuurd.
